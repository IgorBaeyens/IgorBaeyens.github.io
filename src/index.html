My work proces

Everything will be done by me in a period of 1 to 2 months. This means modeling the character, UV unwrapping it which allows for the texturing of the character, then texturing itself which is giving the character color, rigging the character so it can move, shape keys for the expressions and finally converting the character so it can be used as a Vtuber model.



I will send updates when I finish:

  • modelling the base
  • modelling the outfit and hair
  • texturing
  • the expressions
  • converting to vrm
